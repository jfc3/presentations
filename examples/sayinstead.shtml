Testing of CSS3 Property “Say-Instead”

This page was created to to be able to test that the proposed CSS3 property “Say-Instead” is working with Firefox extension “Fire Vox”, which is a free screen reader and any other screen readers. The “Fire Vox” screen reader will allow you to listen to the difference between the quotes without and then with the "say-instead" CSS.
