Improved Accessible Report

The report below shows each of the libraries fiscal year, quarter, last updated date, and an indicator of whether the total of all the books, magizines, CDs, DVDs, etc. are greater than zero.

The reason this report was set up was to allow the main library staff to see which libraries have correctly put in their inventory information quarterly, compared to those that either did not put in a record or just created a zero filled record.

Each library has it's name and new fiscal year visually list once, but it is actually just being displayed of screen so screen readers can still read the information. If the total is greater than zero we do not display the word “YES” so not to visually clutter the report with to much data. We only display the “Total > Zero” fields on the screen that are not greater than zero as “No” and highlight and bold them so they stand out.
